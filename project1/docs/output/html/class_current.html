Context class (part of the State design pattern).

Stores the data structures that hold the information provided in the files.

◆ filesParsed

uint8_t Current::filesParsed
private

Unsigned 8-bit integer where each of the the 4 LSbits (Least Significant bits) correspond to one of the 4 files that need to be parsed.

Works as a boolean array.

Note
Other functions related to menu execution frequently use the condition filesParsed == 0x0F. If the member variable has that value, it means that the 4 LSbits are set (00001111 == 0x0F), which means that all files have been read.
