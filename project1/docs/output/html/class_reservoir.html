Detailed Description

Type of Location in a water supply network (Graph class).

Each (Water) Reservoir object is represented as a vertex in the graph. These vertexes provide water to the entire network and have a maximum delivery capacity of X m3/sec - they are the "start nodes" of this project's graph (they don't have any incoming edges).
